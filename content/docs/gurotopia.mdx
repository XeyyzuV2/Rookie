# Gurotopia

A lightweight, actively maintained Growtopia Private Server (GTPS) written in modern C/C++.

## Overview

Gurotopia is an open-source Growtopia Private Server — clean code, original work, not a leaked fork. Built with:

- **Modern C++** — Clean, readable, maintainable code
- **Lightweight** — Compiler + SSL + SQL under 1GB
- **Active maintenance** — Regular updates and community support
- **Cross-platform** — Windows 10/11 and Linux (Debian, Ubuntu, Arch)
