# xeyyzu.dev

Personal portfolio and documentation hub for [Xeyyzu](https://xeyyzu.dev) — open-source projects, private servers, modding tools, and collaborative development.

Built with [Next.js](https://nextjs.org), [Fumadocs](https://fumadocs.vercel.app), and [Tailwind CSS](https://tailwindcss.com).

## Getting Started

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000).

## Projects

- **Gurotopia** — Lightweight Growtopia Private Server in modern C/C++
- **Rookie** — Documentation website for Gurotopia
- More coming soon...
